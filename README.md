# Library-Management-System
📚 Library Management System (C++)

This project is a console-based Library Management System developed in C++ as part of my early coursework in Computer Science. It demonstrates fundamental programming concepts including object-oriented design, file handling, and basic data management.

The system allows users to manage books and student records efficiently through a simple command-line interface. Core functionalities include adding and displaying books, registering students, issuing and returning books, and maintaining persistent data storage using CSV files.

The project is structured using classes such as Book, Student, and Library, showcasing encapsulation and modular design. File handling is implemented using standard C++ libraries to ensure that data is saved and loaded between program executions.

This project reflects my understanding of:

Object-Oriented Programming (OOP)
File Handling (CSV-based storage)
Data structures using arrays
Basic system design and user interaction
🚀 Key Features
Add, view, and manage books
Register and manage students
Issue and return books
Persistent storage using file handling
Simple and user-friendly CLI interface
🎯 Purpose

This project was built to strengthen my programming fundamentals and gain hands-on experience in designing a real-world application at an early stage of my Computer Science journey.
